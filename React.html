¿Qué son clases en React?

En React, las clases son un concepto fundamental utilizado para definir componentes. Los componentes en React pueden ser creados como funciones o como clases. Las clases en React son una forma de definir componentes utilizando la sintaxis de clases de JavaScript. Antes de la introducción de los "hooks" en React (a partir de la versión 16.8), las clases eran la principal manera de definir componentes con estado (stateful components) y componentes con métodos de ciclo de vida (lifecycle methods). Sin embargo, con la llegada de los hooks, se proporcionaron formas alternativas y más flexibles de gestionar el estado y el ciclo de vida de los componentes utilizando funciones.

Principales Eventos de React

  • onClick: Ocurre cuando se hace clic en un elemento, como un botón.
  • onChange: Se utiliza para elementos de formulario y ocurre cuando su valor cambia.
  • onSubmit: Se activa cuando se envía un formulario.
  • onMouseOver y onMouseOut: Ocurren cuando el cursor se mueve sobre o sale de un elemento.
  • onKeyDown, onKeyUp y onKeyPress: Ocurren al presionar o liberar teclas en el teclado.
  • onFocus y onBlur: Ocurren cuando un elemento recibe o pierde el foco.
